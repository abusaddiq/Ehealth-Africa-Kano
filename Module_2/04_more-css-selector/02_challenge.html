THE ANATOMY OF A RULE

every rule uses only three ingredients: Selectors, Properties, and Value.

Selector

  • identifies the type of content you want to format
  • It allows the browser to hunt down all the parts of a web page that match the selector

Property

  • identifies the type of formatting you want to apply
  • It is where you choose whether you want to change colors, fonts, alignment, or something else

value

  • It sets a value for the property defined
  • It is where you bring it all home. For example, if your property is color, the value could be light blue or a queasy green
